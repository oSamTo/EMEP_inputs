\subsection{PROCESSING FLOW}\label{processing-flow}

The processing in Figure \ref{fig:fig-proc1} shows total emissions as
taken from the maps (inv\_spatial), the inventory table (inv\_table),
the scaled maps (spatial\_scaled), and the area aggregated totals using
the mask.

N.B. the map used is not necessarily the same as the emissions year
(e.g.~MapEire from 2019).

\begin{figure}[h]

{\centering \includegraphics[width=0.6\linewidth]{../outputs/SCOTDOM/SGS2/EMEP4UKv5.0/inv2023/UKEIRE/annual/TPannual_allISO/plots/e2021/nox_UKEIRE_2021emis_2021map_2023inv_2_UKINVBAR} 

}

\caption{Emissions processing from source to area-scaled totals.}\label{fig:fig-proc1}
\end{figure}

Total nox in UK \& Eire, processed: 873 kt + 100 kt = 973 kt. Spatial
breakdown :

\begin{itemize}
\item
  UK = 566.1 kt
\item
  Eire = 93.7 kt
\item
  SEA = 32.7 kt
\item
  Outwith 10km = 280.4 kt
\end{itemize}

\textbf{Overall: Of 973 kt processed, 692.6 kt are input to .nc file,
and 280.4 kt are masked (dropped).}

Figure \ref{fig:fig-proc2} shows the data progressing to NetCDF inputs -
`outwith' data (data outside 10km buffer) is dropped.

\begin{figure}[h]

{\centering \includegraphics[width=0.6\linewidth]{../outputs/SCOTDOM/SGS2/EMEP4UKv5.0/inv2023/UKEIRE/annual/TPannual_allISO/plots/e2021/nox_UKEIRE_2021emis_2021map_2023inv_3_UKNCBAR} 

}

\caption{Area-scaled toals to NetCDF input data.}\label{fig:fig-proc2}
\end{figure}

Emissions coming from processed inventories to NetCDF inputs; `outwith'
emissions (emissions outside 10km sea buffer) are masked and not used
for UK/Eire (replaced by EMEP-EU).
